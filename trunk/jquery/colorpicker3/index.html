JQuery Simple Color Read Me

Click on a textbox to pop out the color picker. Below each example you will see the JS code needed to duplicate the desired effect. More options can be seen at the bottom of the page. The script has been tested with JQuery version 1.2.1..Enjoy! See the bottom for support details.

Demo Number One

  • Autocloses.
  • Cursor is a pointer.
  • Background is colored.

Code:
$("#one").addColorPicker({'colorBg':'yes', 'cursor':'pointer',});

Demo Number Two

  • Closes when clicking on 'Close'.
  • Cursor is a pointer.
  • Background is colored.

Code:
$("#two").addColorPicker({'colorBg':'yes', 'cursor':'pointer','autoClose':'no'});

Demo Number Three

  • Closes when clicking on 'Close'.
  • Cursor is a pointer.
  • Background is colored.
  • Includes a callback function that alerts you of the color picked.

Code:
$("#three").addColorPicker({'colorBg':'yes', 'cursor':'pointer','autoClose':'yes', 'callback': function(c){alert(c);}});

Other options for the Color Picker include:

  • width: default value is 36
  • showCode: default value is 'yes' (which means that it will show the HEX code of the color in the textbox)
  • bgColor: default value is '#000000' (this sets the background color of the box, change to your own #HEX color code)
  • closeText: default value is 'Close'
