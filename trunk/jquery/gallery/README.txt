----------------------------------------------
README
----------------------------------------------

These files are part of a NETTUTS tutorial. Below are explanations of each of the files and folders.

----------------------------------------------


pictures/
----------------------------------------------
This is where your fullsized pictures go, sized 530x272 pixels. Number the pictures 1.jpg, 2.jpg, etc.

thumbs/
----------------------------------------------
This is where your thumbnails go, sized 111x57 pixels. Number the pictures 1.jpg, 2.jpg, etc. Make sure they match the order of the pictures in the pictures/ folder.

index.html
----------------------------------------------
This is the main HTML file.

down.jpg, up.jpg, right.jpg, left.jpg
----------------------------------------------
These are graphics for the four directional arrows used in the gallery.

mootools.js
----------------------------------------------
This contains the full MooTools library, including the Core and More sections.

main.js
----------------------------------------------
This contains all of the JavaScript used to make the gallery work.

main.css
----------------------------------------------
This contains all of the styles for the gallery.
